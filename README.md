# 📦 Система обліку товарів на підприємстві

Дипломний проект — веб-інформаційна система обліку товарів, розроблена на Laravel 13.

## 🛠 Технології

- **PHP** 8.3
- **Laravel** 13.x
- **Livewire** 3.x
- **Flux UI** (компоненти інтерфейсу)
- **SQLite** (база даних)
- **Vite** (збірка assets)
- **Pest** (тестування)

## ⚙️ Функціонал

- 🔐 Авторизація (адмін / менеджер)
- 📦 Облік товарів (CRUD)
- 🗂 Категорії товарів
- 🏭 Склади
- 📊 Рух товарів (прихід / витрата)
- 📈 Звіти та залишки

## 🚀 Встановлення та запуск

### Вимоги
- PHP 8.3+
- Composer
- Node.js 18+
- Laragon або інший локальний сервер

### Кроки

**1. Клонувати репозиторій**
```bash
git clone https://github.com/YOUR_USERNAME/diplom.git
cd diplom
```

**2. Встановити залежності**
```bash
composer install
npm install
```

**3. Налаштувати середовище**
```bash
cp .env.example .env
php artisan key:generate
```

**4. Створити базу даних і заповнити тестовими даними**
```bash
php artisan migrate --seed
```

**5. Запустити сервери**

Термінал 1:
```bash
php artisan serve
```

Термінал 2:
```bash
npm run dev
```

**6. Відкрити в браузері**
http://localhost:8000
